About

This is a Single-Page Login/Logout Application with three views:

  • A welcome view
  • A login view (form)
  • A private view (visible to logged-in users only)

The server is a standalone service implemented using Node.js


Requirements

  • Browser supporting HTML5.
  • Cookies must be enabled on the browser.
  • Javascript must be enabled on the browser.

Installation

  • Download and install nvm

    curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.33.0/install.sh | bash

  • Install the latest version of Node.js

    nvm install node

  • Use the latest version

    nvm use node

  • Install the dependencies

    nvm install

  • Starting the server on port 3000

    nvm start

Exceptions/End Cases

  • APIs Authentication: APIs are tested for authentication.
  • Login Button Disabled: Once user clicks the login button, it is disabled till the status is "Logging In".
  • Cookie Check: If cookie is set/user is logged in, the user is redirected to the private view if login page is accessed. If the cookie is not set/user is logged in, user gets an UNAUTHORIZED error(can be redirected to home/login view in future).
